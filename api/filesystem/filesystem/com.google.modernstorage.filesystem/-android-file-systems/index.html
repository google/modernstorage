Android wrapper around the FileSystems factory methods.

This class handles the creation of a ContentFileSystemProvider as the default file system provider for content:// scheme Uris. It also supports registering alternative schemes by calling AndroidFileSystems.installProvider.

This class automatically calls through to FileSystems if it cannot handle the request.
